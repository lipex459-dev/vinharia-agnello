## Vinharia Agnello

## Sobre o Projeto
A Vinharia Agnello é um site fictício desenvolvido com o objetivo de simular a presença online de uma empresa especializada na venda de vinhos. O projeto foi criado para fins acadêmicos, com foco na prática de desenvolvimento web utilizando HTML.

O site apresenta uma proposta moderna e elegante, refletindo a sofisticação e tradição do mercado de vinhos.

## Objetivo
O objetivo deste projeto é desenvolver um site institucional para uma vinícola fictícia, aplicando conceitos de:

## Estruturação HTML
Organização de conteúdo
Criação de formulários
Boas práticas de desenvolvimento web

## Funcionalidades
O site conta com as seguintes seções:
Home
História
Catálogo
Contato

## Tecnologias Utilizadas
HTML
CSS
